Rules to condsider:

  • Only use "Open Sans". Do not set new font-family or sets. Either allow the class to inherit, or override with "'Open Sans', Helvetica, Arial, sans-serif". (Note: "Open Sans light" does not call Google Fonts, therefore, the next font referenced in the value will load. To call a weight, use font-weight selector,+ values below.)
  • Only the following font-weights are available and loaded for Open Sans: 300, 400, 600, 700, 800. Do not vary from these values. Do not use "bold" or "bolder".
  • Never use Secondary color (#0076BE) for anything other than an item in which suggests interactivity. For instance, do not set a header in this color unless the header is linked.
  • Colors Lily, Tomato, Citrus, Daisy, and Leaf should very rarely be used.
  • Do not create a new color. Variations of colors can be used by adding an alpha value (0.01-1) » rgba(#,#,#,#).
  • Do not center headers unless it is integral to the optical anchoring of more than one element on the page.
